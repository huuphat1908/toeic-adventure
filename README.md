## Yêu cầu thiết bị
- Android
  - minSdkVersion: 21.0
  - targetSdkVersion: 32

- Công nghệ
  - Ngôn ngữ: Java
  - Backend Platform: [Node.js](https://nodejs.org/en/)
  - Database: [MongoDB Atlas](https://www.mongodb.com/cloud/atlas)
